# blog-api

This is a backend api that was made for The Odin Project. It contains models for users, comments, and blog posts. It is used as the backend of two other websites.

# Tools Used

-   Express
-   MongoDB
-   Mongoose
-   BcryptJS
-   PassportJS
-   cors
-   dotenv
-   JWT - JSON Web Token
-   npm
-   git
